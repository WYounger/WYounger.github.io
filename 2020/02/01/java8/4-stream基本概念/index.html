4.Stream基本概念

Posted on 2020-02-01 | In java8

首先了解以下流的概念

从支持数据处理操作的源生成的元素序列

流的特性的如下:

  • 元素序列。就像和集合一样，区别在与集合强调数据的存储，而流强调数据的计算
  • 源。流需要源，通常来自集合和数组
  • 数据处理操作。filter,map,reduce,find,match,sort,limit
  • 流水线。很多流操作会返回流，所以可以进行流水线操作
  • 内部迭代。不像迭代器的显示迭代元素，流的迭代操作在背后进行

例子

1,2,3,4属于流的中间操作(返回stream的操作)，5属于终端操作

常见的中间操作

操作 操作对象 函数描述符
filter Predicate T -> boolean
map Function<T,R> T -> R
limit
sorted Comparator (T,T) -> int
distinct

常见的终端操作

操作 描述
forEach Consumer<? super T>
count 返回流中的元素个数
collect 把流归约成一个集合,如List,Map
